Petunjuk Penggunaan

Thermal Printer Bluetooth - EPSON EP5821

Aplikasi Utama Mode Demo

Persiapan Awal

1 Pastikan Printer Siap
  • EPSON EP5821 dalam kondisi ON
  • Kertas thermal 58mm terpasang dengan benar
  • Baterai printer terisi atau terhubung ke charger
  • LED indicator menunjukkan printer ready
2 Aktifkan Bluetooth
  • Aktifkan Bluetooth di smartphone/device
  • Pastikan device dalam jangkauan (maksimal 10 meter)
  • Untuk smartphone Android, aktifkan juga Location/GPS
3 Mode Pairing Printer
  • Tekan dan tahan tombol Bluetooth di printer
  • LED Bluetooth akan berkedip menandakan mode pairing
  • Printer siap untuk ditemukan oleh device
Penting: Jangan pair printer melalui pengaturan Bluetooth sistem. Gunakan tombol "Hubungkan Printer" di aplikasi web ini.

Menghubungkan Printer

1 Buka Aplikasi

Akses aplikasi melalui browser yang didukung. Pastikan JavaScript aktif.

2 Klik "Hubungkan Printer"

Tombol biru di bagian atas aplikasi. Browser akan meminta permission untuk akses Bluetooth.

3 Pilih Device

Cari dan pilih "EP5821" atau nama printer yang muncul di daftar.

4 Konfirmasi Koneksi

Status akan berubah menjadi "Terhubung" dan tombol menjadi "Putuskan Koneksi".

Log yang normal: [10:30:15] Mencari perangkat Bluetooth... [10:30:18] Perangkat ditemukan: EP5821 [10:30:20] Menghubungkan ke GATT server... [10:30:22] Berhasil terhubung ke printer!

Cara Penggunaan

Cetak Teks Sederhana

1 Pilih tab "Cetak Teks" dan masukkan teks yang ingin dicetak
2 Atur alignment (Kiri/Tengah/Kanan) dan ukuran teks
3 Klik "Cetak Teks" dan tunggu hingga proses selesai

Cetak Struk Belanja

1 Isi data toko (nama, alamat, kasir) di tab "Cetak Struk"
2 Tambah item belanja dengan tombol "Tambah Item"
3 Atur metode pembayaran dan jumlah yang dibayar
4 Klik "Cetak Struk" untuk mencetak struk lengkap

Test Print

Gunakan berbagai tombol test untuk memvalidasi fungsi printer:

  • Test Basic: Print sederhana dengan informasi dasar
  • Test Alignment: Test rata kiri, tengah, kanan
  • Test Size: Test berbagai ukuran font
  • Test Barcode: Print barcode sample
  • Potong Kertas: Manual paper cut

Troubleshooting

‚ùå Printer Tidak Terdeteksi

  • Pastikan printer dalam mode pairing (LED Bluetooth berkedip)
  • Restart printer dan coba lagi
  • Periksa jarak antara device dan printer
  • Refresh halaman web dan coba koneksi ulang
  • Pastikan browser mendukung Web Bluetooth API

‚ùå Koneksi Terputus

  • Periksa level baterai printer
  • Pastikan printer tidak dalam mode sleep
  • Jaga jarak koneksi dalam radius optimal (1-5 meter)
  • Hindari interferensi dari device Bluetooth lain

‚ùå Hasil Print Tidak Sesuai

  • Pastikan menggunakan kertas thermal 58mm
  • Periksa kualitas kertas (tidak expired)
  • Bersihkan head printer jika hasil print pudar
  • Pastikan kertas terpasang dengan benar

‚ùå Browser Tidak Support

  • Gunakan Chrome/Edge/Opera versi terbaru
  • Aktifkan "Experimental Web Platform features" di chrome://flags
  • Pastikan HTTPS atau localhost (Web Bluetooth requirement)
  • Restart browser setelah mengubah pengaturan
Debug: Periksa log aktivitas di bagian bawah aplikasi untuk informasi error detail.

FAQ

Q: Apakah aplikasi ini bisa digunakan offline?

A: Ya, setelah pertama kali dibuka, aplikasi akan tersimpan di cache browser dan bisa digunakan offline.

Q: Bisakah mencetak logo atau gambar?

A: Saat ini hanya mendukung teks dan barcode. Fitur gambar akan ditambahkan di versi mendatang.

Q: Apakah data aman?

A: Ya, semua data diproses secara lokal di browser. Tidak ada data yang dikirim ke server eksternal.

Q: Printer lain selain EP5821 bisa digunakan?

A: Ya, printer thermal lain yang mendukung ESC/POS dan Bluetooth SPP umumnya compatible.
