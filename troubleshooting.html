Troubleshooting Guide

Panduan mengatasi masalah koneksi Bluetooth Printer

Masalah HTTPS

Web Bluetooth API membutuhkan HTTPS!

Browser modern mengharuskan website menggunakan HTTPS untuk mengakses Web Bluetooth API demi keamanan.

Solusi:

  1. Gunakan HTTPS: Host website di server yang mendukung HTTPS (SSL/TLS)
  2. Localhost: Jalankan di localhost atau 127.0.0.1 (tidak memerlukan HTTPS)
  3. Development Server: Gunakan tools seperti Live Server dengan HTTPS enable

Printer RPP02 Tidak Terdeteksi

Langkah troubleshooting:

  1. Pastikan pairing: RPP02 sudah paired di pengaturan Bluetooth Android
  2. Mode discoverable: Aktifkan mode discoverable pada printer
  3. Jarak: Pastikan jarak dekat (< 5 meter) dari printer
  4. Battery: Pastikan printer memiliki daya yang cukup
  5. Browser support: Gunakan Chrome/Edge versi terbaru

Cara Mengaktifkan Mode Discoverable RPP02

Untuk printer RPP02:

  1. Tekan dan tahan tombol Power + Feed bersamaan
  2. Tunggu hingga LED berkedip biru atau indikator Bluetooth aktif
  3. Printer sekarang dalam mode discoverable
  4. Coba koneksi kembali dari aplikasi web

Mode Debug

Gunakan tombol "Debug Connect (RPP02)" pada halaman utama untuk:

  • Melihat informasi sistem detil
  • Menguji berbagai filter pencarian perangkat
  • Log detil proses koneksi
  • Identifikasi masalah spesifik
